10 главных выводов

2 из 5 Tier 1 поглощены в 2025: Paradox → Workday ($1B), Moveworks → ServiceNow ($2.85B). Консолидация платформ — основной путь выхода.

Заявления об автономности систематически завышены: только Paradox (scheduling) и Leena AI (тикеты) обеспечивают измеримую end-to-end автоматизацию.

Per-employee ценообразование побеждает: Leena AI, Eightfold и Moveworks берут оплату за сотрудника, а не за HR-лицензию.

Ни один стартап не предлагает on-prem: все 10 — SaaS-only. Критический GAP для российского enterprise и регулируемых индустрий.

Rippling — аномалия: единственный PLG, $570M+ ARR, 10 255 отзывов на G2, оценка $16.8B.

Beamery подаёт сигналы бедствия: два раунда сокращений в 2023 (12% + 25%), $223M привлечено при ACV $220K-$580K.

Сертификаты не спасают от утечек: Paradox имел SOC 2 + ISO 27001, когда тривиальная уязвимость раскрыла 64M записей кандидатов.

Eightfold — лидер по compliance: FedRAMP Moderate + DISA IL4 + ISO 42001, но G2 рейтинг (8.3/10) самый низкий среди Tier 1.

Multi-LLM архитектуры — новый стандарт: Leena AI открыто использует WorkLM + Gemini + Claude + OpenAI динамически.

Самый конкурентный ICP: enterprise 1000-10000 сотрудников с CHRO как покупателем — 4+ стартапа борются за одного клиента.

Ни одна компания не обеспечивает истинную автономию для сложных HR-решений (промоции, компенсации, орг. дизайн).

2. Какая монетизация побеждает в enterprise

Побеждает: WORKFORCE-BASED
Выравнивает доход вендора с headcount клиента. Естественный рост при расширении. Leena AI, Eightfold, Moveworks.
Сохраняется: SEAT-BASED
Для инструментов используемых только HR-командой. Ограничивает TAM. Beamery, Lattice, Phenom.
Редкость: OUTCOME-BASED
Pay-per-resolution / pay-per-hire пока не масштабировался. Только Workera приближается с per-assessment.

3. Самый конкурентный сегмент

Overlap по ICP: Enterprise 1K-10K сотрудников, покупатель CHRO

4. PLG vs SLG: эволюция

PLG процветает
Для HR-инструментов используемых HR-командой (Lattice, Rippling). Покупатель = пользователь. Self-serve → trial → adoption.
SLG-ENT доминирует
Для AI-платформ требующих интеграцию, security review, IT-involvement. 8 из 10 — SLG-ENT.

Мёртвая зона: SLG-MID — сочетает стоимость enterprise-продаж с бюджетными ограничениями mid-market. Самая слабая позиция.

Критические GAP-ы рынка

On-Prem развёртывание

0 из 10 стартапов предлагают on-prem. Российский enterprise (гос., финансы, оборона) требует суверенности данных. Это главный моат.

Интеграция с 1С:ЗУП

Западные стартапы интегрируются с Workday/SuccessFactors. Для RU рынка нужна нативная интеграция с 1С:ЗУП, БОСС-Кадровик, SAP HCM Russia.

Локальные LLM

Leena AI использует GPT + Claude + Gemini. Для RU: YandexGPT/GigaChat + Mistral/Llama open-source на собственной инфраструктуре.

Регуляторика 152-ФЗ

SOC 2 / GDPR не релевантны. Нужен 152-ФЗ compliance, требования ФСТЭК для гос. органов, локализация данных.

Русский NLP для HR

Западные модели underperform на русской HR-терминологии. Нужен fine-tune на российском трудовом кодексе, кадровых документах, ЛНА.

Skills taxonomy для RU

Нет российского аналога talent graph Eightfold (1.6B профилей). Возможность построить на данных hh.ru, SuperJob, HeadHunter.
